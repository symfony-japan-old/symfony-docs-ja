Regex
=====

値が、正規表現にマッチするかどうかを検証します。

.. code-block:: yaml

    properties:
        title:
            - Regex: /\w+/

オプション
----------

* ``pattern`` (**デフォルト**, 必須): 正規表現パータン
* ``match``: パターンがマッチしなければならないまたは、マッチしてはならないかどちらか。
  デフォルト: ``true``
* ``message``: 検証に失敗した場合のエラーメッセージ
